##  Key Learnings

Through executing and explaining these colabs, the following core deep learning concepts were reinforced:

- **Activation functions** control gradient flow — choosing the right one prevents vanishing gradients
- **Optimizers** differ significantly in convergence speed — Adam adapts to sparse gradients most effectively
- **Modern CNN architectures** (ResNet, EfficientNet) solve degradation via skip connections and compound scaling
- **Hyperparameter tuning** has a compound effect — learning rate and batch size together determine training stability
- **CNN fundamentals** (conv filters, pooling, BatchNorm) are the foundation of all vision models
- **Evaluation metrics** beyond accuracy (F1, ROC-AUC) are essential for real-world imbalanced datasets
